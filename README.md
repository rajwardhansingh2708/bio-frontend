Overview



This is the frontend interface for the Visitor Analytics \& Logging System.



It is a responsive, animated landing page that displays social media links in a modern bento-grid style UI. The frontend integrates with a backend analytics service to silently log visitor metadata.







\## Features



\- Animated gradient background

\- Glassmorphism card UI

\- Responsive grid layout

\- Social media icon branding

\- Silent backend visitor tracking

\- Mobile-friendly design
